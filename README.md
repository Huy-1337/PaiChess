# PaiChess

Tauri + React chess GUI scaffold with:

- OS-aware engine URL resolution for Stockfish and Dragon.
- UCI-style move classification helpers (Best/Good/Inaccuracy/Miss/Mistake/Blunder).
- Puzzle CSV line parser targeting Lichess puzzle dumps.
- React chessboard integration with `chess.js` game state.

## Run

```bash
npm install
npm run dev
```

## Test

```bash
npm test
```
